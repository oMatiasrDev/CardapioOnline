# 🍝 Cardápio Online - Massas Artesanais

Projeto de um site de cardápio digital desenvolvido com HTML, CSS e JavaScript, com foco em responsividade, interatividade e integração com WhatsApp para pedidos.

---

## 🚀 Funcionalidades

- 📋 Exibição de cardápio online
- 🍝 Listagem de pratos com imagens e preços
- 📱 Pedido direto via WhatsApp
- 📲 Integração com redes sociais
- 📌 Navegação por seções (Home, Sobre, Cardápio, Contatos)
- 📱 Menu responsivo (hambúrguer)
- ⚙️ Interatividade com JavaScript (abrir/fechar menu)

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap Icons
